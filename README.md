# KeyRing #

## Introduction ##

KeyRing is the service that deals with the password manager,

This project is mostly being used for me to learn different DevOps and cloud technologies like terraform, docker, and gcp.

## EndPoints ##

* /keyring
  * Get
  * Post
  * Patch
  * Delete

* /health
  * Get

## Technologies Used ##

### Currently ###

* GCP CloudSQL
* Docker
* JWT

## Planned ###
